## User Story

- As a user from ages 12+ are able to use this game to have fun while learning JavaScript. It's and resource and a study tool
- As a user they will be able to test their skills with a Q&A bank that has simple questions to give a small sample of how to form functions and arrays.
- As a user they will only have to click in the area designated to see the answer.
